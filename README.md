blaze_test
==========

Experimenting with Meteor's Blaze templating engine, sans-meteor.


### Getting Started

Install Blaze via bower:

```
bower install
```

Use http-server to preview the app:

```
npm install http-server -g
cd blaze_test
http-server -p 9000
```

And visit (localhost:9000/lib)[http://localhost:9000/lib]
